# 🩺 Diabetes Prediction using Machine Learning

This project predicts whether a patient has diabetes using machine learning based on medical input features.

## Dataset
- diabetes.csv

## Files
- diabetes_prediction.ipynb – Data analysis, model training & evaluation
- diabetes_model.pkl – Saved trained model

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## Workflow
1. Data preprocessing
2. Exploratory Data Analysis
3. Model training
4. Model evaluation
5. Model saving using Pickle
